# RISC-V Instruction Set Explorer

This project is part of the RISC-V Mentorship Coding Challenge. It parses the RISC-V instruction set from public datasets, groups them by their extension tags, and cross-references them against the official RISC-V ISA manual.

## Getting Started

1. Clone this repository.
2. Set up a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Project Structure
- `src/`: Contains the main source code.
